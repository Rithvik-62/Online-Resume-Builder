# Online Resume Builder

A simple web-based application that helps users create professional resumes quickly using an easy form interface.

# 🚀 Project Overview

The Online Resume Builder allows users to:
Enter personal, educational, and skill details
Generate a professional resume instantly
Edit and update resume information easily
Save time compared to creating resumes manually

# 🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
Tools: XAMPP, VS Code

# ✨ Main Features
User-friendly interface
Resume generation system
Educational & skills section
Work/project details section
Easy form-based resume creation
Database connectivity using MySQL

# ⚡ Problem We Solved
Creating resumes manually in Word or other software can:
Take a lot of time
Be difficult for beginners
Cause formatting problems
Make editing harder
Our project simplifies the process by generating resumes through an online form system.

# 🧩 Challenges We Faced
1. Database Connection Issues
We faced problems connecting PHP with MySQL.<br>
Solution:
We checked XAMPP configuration, database settings, and fixed connection errors step by step.

2. Form Data Handling
Handling large amounts of user data from forms was difficult.<br>
Solution:
We organized the form handling code properly using PHP.

3. Resume Layout Design
Making the resume look clean and professional was challenging.<br>
Solution:
We improved the CSS styling and tested different layouts.

4. Team Coordination
Managing tasks in a group project was not easy.<br>
Solution:
We divided the work into frontend, backend, and documentation tasks.
